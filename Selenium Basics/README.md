# Selenium Basics - Digital Nurture 5.0

## Description
This repository contains Selenium automation exercises completed as part of Cognizant Digital Nurture 5.0.

## Topics Covered
- Selenium WebDriver
- Locators
- Explicit Wait
- Fluent Wait
- Sleep vs Wait
- PyTest
- HTML Reports
- Page Object Model (POM)

## Project Structure

```
pages/
tests/
conftest.py
setup_test.py
wait_demo.py
clickable_wait.py
fluent_wait.py
sleep_vs_wait.py
locators_waits.py
```

## Test Execution

```bash
python -m pytest tests -v
```

## Generate HTML Report

```bash
python -m pytest tests --html=report.html --self-contained-html
```
